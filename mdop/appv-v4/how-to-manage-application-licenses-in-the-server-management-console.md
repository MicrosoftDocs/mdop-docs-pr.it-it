# Come gestire le licenze dell'applicazione in Server Management Console


Application Virtualization Server Management Console è l'interfaccia usata per gestire la piattaforma di virtualizzazione dell'applicazione. Da esso è possibile aggiungere, rimuovere, configurare e controllare i gruppi di licenze applicative.

**Importante**  Se l'impostazione dell'origine dell'applicazione client App-V (ASR) è configurata in modo da usare qualsiasi tipo di origine di flusso diversa da quella del server di gestione, ad esempio un server di flusso, un server IIS o un file server, il server di gestione non è in grado di applicare i criteri di licenza.
